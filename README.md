# Tushare 数据接口与积分门槛决策辅助工具

> 帮助 Tushare 用户快速查询各数据接口所需的积分门槛，支持按积分档位筛选、搜索和分类浏览。

## 在线访问

**https://WayhomeZ.github.io/tushare_credits_tool/**

## 功能特性

- 📊 **积分档位筛选**：按 1000~8000+ 积分区间快速查看可用接口
- 🔍 **搜索功能**：支持按接口代码、名称、描述搜索
- 📂 **分类浏览**：完全对齐 Tushare 官网侧边栏分类结构
- ✨ **新增高亮**：切换积分档位时，新增解锁的接口会高亮显示
- 🔗 **文档直达**：每个接口都附带官方文档链接

## 使用说明

1. 选择积分档位卡片，查看该档位及以下所有可用接口
2. 点击分类标题可展开/收起该分类下的接口列表
3. 在搜索框输入关键词，自动过滤并展开匹配的分类
4. 紫色高亮行表示当前档位**新增解锁**的接口
